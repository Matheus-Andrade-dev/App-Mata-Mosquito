Mata Mosquito – Jogo em HTML, CSS e JavaScript

Um jogo simples e divertido desenvolvido com HTML, CSS e JavaScript, onde o objetivo é eliminar os mosquitos que aparecem aleatoriamente na tela antes que o tempo acabe.
O projeto foi criado para praticar lógica de programação, manipulação do DOM, eventos, temporizadores e posicionamento dinâmico de elementos na página.

Funcionalidades

1- Geração aleatória de mosquitos em diferentes posições.

2- Três níveis de dificuldade.

3- Contagem regressiva de tempo.

4- Perde vidas ao deixar mosquitos escaparem.

5- Interface simples e responsiva.
